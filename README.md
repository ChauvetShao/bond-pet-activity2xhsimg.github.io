# BOND 内容运营工具箱 (BOND Content Operations Toolbox)

本工具箱是专为 **BOND 宠物社区** 内容团队打造的创作利器，旨在通过 AI 赋能和模块化设计，提升小红书、公众号等社交平台的图文创作效率。

---

## 🛠 核心工具集

### 1. 宠物活动排版工具 (`pet-events.html`)
专为每周宠物活动资讯设计，将繁杂的活动信息一键转化为高颜值的小红书合集图。

*   **智能解析**：支持 AI 批量解析群公告、博文原文，自动提取时间、地点、费用等结构化数据。
*   **海报对位**：支持上传 ZIP 压缩包，按文件名自动匹配活动详情图。
*   **高清导出**：16 倍超高清分辨率输出，完美适配小红书画质。
*   **一键文案**：自动生成适配小红书的排版文案，包含 Emoji 和关键信息。

### 2. 模块化图文排版工具 (`xhs-layout.html`)
自由度极高的模块化编辑器，适用于深度内容创作、攻略分享及公众号长文。

*   **自由组合**：提供封面、图文、表格、清单、活动卡片、互动引导等多种预设模块。
*   **AI 智能辅助**：
    *   **✨ AI 一键排版**：粘贴文章原文，AI 自动规划画布数量并填充内容模块。
    *   **🎨 AI 批量配图**：一键为所有模块生成精美的 CogView-3 插画配图。
*   **可视化编辑器**：
    *   **表格编辑**：类 Excel 的直接输入体验，支持动态增删行列。
    *   **实时预览**：所见即所得，支持导出 PNG 图片及公众号 HTML 代码。

## 🚀 快速上手

### 1. 环境准备
*   本工具为纯前端实现，无需安装。直接使用现代浏览器（推荐 Chrome 或 Edge）打开 `index.html` 即可。
*   **API Key 配置**：
    1. 进入「图文排版工具」。
    2. 点击侧边栏的 **「设置」** 标签。
    3. 填入您的 [智谱 AI API Key](https://bigmodel.cn/apikey/platform)。
    4. Key 将安全地保存在本地浏览器缓存中。

### 2. AI 创作流程 (图文排版)
1. 切换至 **「AI 排版」** 标签页。
2. 粘贴您的原始稿件，点击 **「✨ 开始 AI 排版」**。
3. 排版完成后，点击 **「🎨 一键生成文章配图」**。
4. 在全屏加载完成后，您将得到一套完整的高质量图文素材。

## 💡 功能亮点

*   **极速响应**：所有编辑操作均在 0.1 秒内实时同步预览。
*   **专注输入**：优化的属性面板逻辑，确保输入过程中不失焦、不闪烁。
*   **数据安全**：支持「保存草稿」功能，将所有内容保存为本地 JSON 文件，随时回载。
*   **跨平台适配**：
    *   **小红书**：导出 1242x1660 标准尺寸 PNG。
    *   **公众号**：一键复制带内联样式的 HTML 代码，直接粘贴至微信编辑器。
