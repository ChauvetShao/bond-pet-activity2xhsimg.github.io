# 小红书宠物活动排版工具使用说明

本工具是一款专为宠物博主和活动组织者设计的单文件、纯前端排版工具。它能够将凌乱的活动信息快速转化为高颜值的小红书笔记图。

## 🚀 核心工作流

建议按照以下顺序使用工具以达到最高效率：

### 1. 数据录入（三选一）
*   **方式 A：表格手动录入**
    在左侧工作区直接填写活动信息。支持直接修改标题、主办方、日期、地点、简介、标签和报名方式。
*   **方式 B：Excel 导入（推荐）**
    点击「导入 Excel」选择您的活动排期表。工具会自动按列名匹配数据。
    *   *注意：如果您的 Excel 包含「海报数据(Base64)」列，图片也会一并恢复。*
      1. 从飞书多维表格下载Excel，并将图片作为附件保存**「请勾选使用索引列名称命名」**https://jcn4qkhe8vt2.feishu.cn/base/TUlgbUF6uaEplts0vMacivatnKf?from=from_copylink
    *   
*   **方式 C：AI 批量解析**
    点击「AI 批量解析」，粘贴原始的文本信息（如群公告、长博文），AI 会自动提取结构化数据并填入表格。

### 2. 海报批量匹配
如果您有大量海报图片，无需一张张上传：
1.  将图片命名为对应的「活动标题」（例如：`狗狗聚会.jpg`）。
2.  将所有图片打包成一个 `.zip` 压缩包。
3.  点击「上传海报 ZIP」，系统会自动完成所有海报的精准对位。

### 3. 定制封面与结尾
*   **封面标题**：在工具栏输入框填写本次笔记的主题（如：北京 4 月领养日合集）。封面图中央会实时显示该标题。
*   **自动预览**：右侧预览区会自动生成第 1 张封面图和最后 1 张结尾图。

### 4. 导出与发布
*   **导出全部 PNG**：点击按钮，系统会以 **16 倍超高清分辨率** 生成整套图片。
    *   `xhs-card-cover.png`：封面图
    *   `xhs-card-x.png`：活动详情图（每张图包含 2 个活动）
    *   `xhs-card-last.png`：结尾图
*   **复制全部文案**：一键获取小红书正文所需的文字排版，包含主办方、日期、地点等详情。

## 💡 功能亮点
*   **超高清输出**：导出分辨率极高，完美适配小红书高清画质。
*   **离线存档**：导出 Excel 时会自动包含图片的 Base64 数据，一份文件即可备份整个活动数据库。
*   **智能排版**：活动海报与内容采用左右交错分布设计，视觉感官更平衡、专业。
*   **即时同步**：左侧任何微小的改动，右侧预览都会在 0.1 秒内同步更新。

## 🛠 配置说明
*   **AI 接口**：默认预设通义千问配置。如需使用，请填写您的 API Key。
*   **运行环境**：无需安装任何软件，直接使用现代浏览器（Chrome, Edge, Safari）打开 `index.html` 即可。
*   **本地素材**：请确保 `img` 文件夹内包含 `bg.png`, `cover-bg.png`, `last.png` 以保证视觉效果完整。
